# Gulp4-project

> Сборка работает на gulp версии 4.0. 

#### Для начала работы yarn

1. ```clone this repo```
2. ```cd path/to/...```
3. ```yarn add global gulp-cli```  
> Установка последней версии Gulp CLI tools

4. ```gulp```

#### Для начала работы npm

1. ```clone this repo```
2. ```cd path/to/...```
3. ```npm install --global gulp-cli```  
> Установка последней версии Gulp CLI tools

4. ```gulp```
